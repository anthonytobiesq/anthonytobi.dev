Open Source Shield

Concept

Modern software relies heavily on open source components. However, not all open source licenses are compatible with proprietary commercial software.

Open Source Shield is a command-line utility built in Go that scans your codebase, identifies all dependencies, and alerts you if there is a licensing conflict (such as a GPL-v3 library in a closed-source product).

This ensures startups remain compliant from day one, avoiding costly legal refactoring down the line.
